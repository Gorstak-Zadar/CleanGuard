# 🔒 CleanGuard

> **File Reputation & Cleanliness Guard** - Multi-source file verdict aggregation for threat detection.

---

## 📋 Overview

CleanGuard aggregates file reputation data from multiple sources to provide comprehensive file verdicts, identifying clean, suspicious, or malicious files through reputation scoring.

---

## 🎯 What It Does

- 🔍 **Multi-Source Reputation** - Aggregates multiple threat intel sources
- 📝 **File Verdicts** - Clean, suspicious, malicious classifications
- 🔄 **Cache Management** - Reputation result caching
- 📊 **Confidence Scoring** - Aggregated confidence levels

---

## 🚀 Usage

```powershell
# Run file reputation check
.\CleanGuard.ps1
```
